Visit the Library with the Manage own role
 

You will find in this page all what you need to know in order to visit the Library with the Manage-own role.


You can navigate to any part of this page through the links below:

  • Add content into the library
  • Delete a content

With Manage-own role, you have been invited to increase the Library with new contents:

  • You are allowed to add contents.
  • Since you might have to modify these contents, you can edit your own contents. But you cannot edit contents created by other members.
  • You can delete only your own contents.

Add content into the library

Wherefrom can I start the "Add content" action?

You will find the "Add content" action under the header "Library", from the list of the services of your interest group. If you just add content to a sub-space of the Library, display this sub-space and use the "Add content" function you can see in the library's menu.



Action "Add content" is available from the library menu

How can I add content?

You open the Add content dialog and browse your system to locate the file you wish to add. Once done, you upload the selected file.
Please be aware that you cannot upload files bigger than 100 Mb.
The bulk-upload feature also has a 20 Mb zip limit.
