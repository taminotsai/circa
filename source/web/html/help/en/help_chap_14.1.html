Presenting information through the information service
 

How can I publish to the information service ?

In a few words

First you create a mini website, locally. Then you will upload its elements to the information service. Lastly, you associate an index file through which you set links to the documents to be browsed on the publishing area.

Creating an information page

Before logging in CIRCABC, you will have designed your own web pages. Then you log in and go to your interest group.

You access the repository of documents to be published by choosing the option "Browse repository" from the option "Information" of the main menu of the interest group.


"Information" allows you to add and publish content

The available actions:

  • You load your html file through command "Add content". We add "EU-gallery.htm", for instance
  • You create a space where you store the files originally placed in the files folder (this contains the documents you have inserted in your web page, as pictures). Our folder "EU-gallery_files" contains 5 pictures.
    So we create the space titled "EU-gallery_files". Then we upload the 5 images into it. A further release of CIRCABC will offer the opportunity to load these documents in bulk

We upload web pages to the "information" service
  • Lastly, you define the index page which links together the documents you have just added. To do so, you click the menu option "Administration" from the information service main page. And opt for link "Edit information properties". You are taken to the page titled "Modify space properties". You put the exact name of your mini website in the field so-called "index page"

Crossing "adapt" means that the information page will be displayed in the same window

You are asked to check or not to check function "Adapt". Checking it you make your information page to be displayed in the same window as CIRCABC. Otherwise, a new window is opened.

Now, you confirm the changes made and close "Edit information properties".

Well, but how can I verify the lay out of my new information page?

  • By clicking the command "Preview" that can be found in the right-hand up corner of the screen, you can watch your information page such as will be seen by users

Modifying an information page

The spaces in the information service look like multilingual spaces in the Library. You can edit even remove them in a same way.

The contents have properties similar to contents in the Library. The same actions apply to them.
