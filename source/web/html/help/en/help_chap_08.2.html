Setting your language preferences
 

This page helps you to work in your favourite language.


You can navigate to any part of this page through the links below:

  • Selecting a language of the user interface
  • Selecting a language for the display of contents

Users can draw their own linguistic profile and specify:

  • the language in which you would like to have the menus displayed;
  • the language in which you would like to have the contents displayed.


Two linguistic preferences

Is the use of these options subject to any condition?

- Any user can change the User Interface language while only users who have made a login can select a Content Language Filter.


Top of the page 

Selecting a language of the user interface

Registering gives you an opportunity to save your favourite User Interface language. But any user, whether or not authenticated, can change the user interface language, temporarily.

  • The language of the user interface is one of the "User options" you are proposed to select and save as filling out the registration form. Later you can change again the interface language via the "Manage your account" command of the main menu.


Registration form: save your favourite User interface language

  • As any user, whether or not authenticated, you can change the User Interface language at any time and from any CIRCABC page. To do so, you use the language menu presented at the top-right corner of screen.


Change the language of the User interface, temporarily

In the above example, we have selected "English". This means that all the menus and pages of your User interface will be worded in English. You can change again the User Interface language at any time.


Top of the page 

Selecting a language for the display of contents

The User options also allow you to read the contents in your favourite language. To do so, you will set a Content Language Filter.



Only authenticated users may apply a "Content Language Filter".


  • You can save your linguistic choice while registering. To do so, you select a Content Language Filter from the "User options" section of the registration form. The multilingual contents are then displayed in the selected language whenever you have authenticated and this, until you select another Content Language Filter.


Saving a Content Language Filter as registering


I have no linguistic preference!
In other words, I wish to read the multilingual contents in any language available. Is it possible?

- Yes, you will simply pick "All languages" out of the list attached to the field "Content Language Filter".




No filter is applied to the content language


This lets you see every linguistic version of a same multilingual content. Moreover, "all languages" is the default filter applied in the registration form.

  • To change the Content Language Filter, you must be authenticated. So, once you have logged in, choose "Manage your account" from the CIRCABC main menu. Scroll down to the "User options" pane, expand "Content language filter" and pick one of the 165 languages listed. You confirm your new choice by pressing on the "Save" button at top of page.


A "Content language filter" is applied

Do not forget that this really makes sense when applied to a group of translations.
