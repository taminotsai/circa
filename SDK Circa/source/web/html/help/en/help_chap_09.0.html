How to deal with searching
You can build more or less complex requests and find any kind of content

CIRCABC-IPM enables you to retrieve any content, stored in any format. The search engine explores the full text of the contents as well as the files names and titles. Notice that the "description" field is not read by the engine. It reads only the last version of every content.


"You seem to tell about textual search. But what about the items which exclusively consist of data other than text?

- No matter, such items are linked to textual metadata (i.e. their name). The search engine explores their name and some properties.
