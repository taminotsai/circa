Visit the Library with the Access role
 

You will find in this page all what you need to know to visit the Library with the Access role.


You can navigate to any part of this page through the links below:

  • How can I view the details of a content?
  • How can I read a multilingual document in my favourite language?
  • How can I download a content?

You dispose of the most limited rights:

  • You are only allowed to read and download contents from the Library.
  • You can choose the language in which you would like to read the multilingual documents.
  • You can view the details of any content but you are unable to modify their properties.

How can I view the details of a content?

You click the "View details" icon (  ) next to the content name in the main page of the Library. By doing so, you display the content properties.
