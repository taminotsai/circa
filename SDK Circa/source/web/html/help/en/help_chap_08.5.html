Displaying the details of a translation
 

Let us focus on the "Properties" and "Multilingual Content Info" areas. Each translation, indeed,

  • includes its own properties;
  • shares properties with every related translation; these shared characteristics are grouped within the "Multilingual Content Info".

You can navigate to any part of this page through the links below:

  • Displaying the properties of a translation
  • Displaying the "multilingual content information"

Displaying the properties of a translation

By clicking the "View Details" icon (  ) which appears next to the translation name, in the "contents" area of the related space, you display the details of a translation.

The details page shows the following areas:

  • Properties
  • Multilingual information
  • Related translations
  • Version History
  • Actions

The "Properties" pane only describes the current translation. Keep in mind that the "Language" field reveals the language of the current translation. It does not refer to the original document (so-called "pivot translation"). Also the version label only refers to the current content.

Displaying the "multilingual content information"

The "Multilingual content information" includes:

  • properties shared by all the translations
  • a reference list of the translations


Displaying "Multilingual Content Info"

The properties listed in "Multilingual Content Information" are common to the whole group of translations. The "pivot translation" field, here, refers to the source of all the translations of the group.
